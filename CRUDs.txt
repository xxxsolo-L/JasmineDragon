1. Продукты (Чай)
Создать новый продукт
POST /products          !complete
Данные: название, описание, цена, категория, количество на складе, изображение, и т.д.

Получить все продукты
GET /products           !complete

Получить продукт по ID
GET /products/:id       !complete

Обновить данные продукта
PUT /products/:id       !complete
Данные: любые параметры продукта (название, описание, цена и т.д.)

Удалить продукт
DELETE /products/:id    !complete

3. Пользователи
Создать нового пользователя (регистрация)
POST /users
Данные: имя, email, пароль и др.        !!!!!!!!!!!!!!

Получить всех пользователей
GET /users (только для админов)

Получить пользователя по ID
GET /users/:id

Обновить данные пользователя
PUT /users/:id
Данные: имя, email и т.д.

Удалить пользователя
DELETE /users/:id (только для админов)

4. Заказы
Создать новый заказ
POST /order
Данные: ID пользователя, список продуктов, адрес доставки, статус, и т.д.       !!!!!!!!!!!

Получить все заказы (для администратора)
GET /orders     !!!!!!!!!!

Получить заказ по ID (админ все, юзер только свои)
GET /orders/:id     !!!!!!!

Обновить статус заказа
PATCH /orders/:id
Данные: статус заказа (например, "обрабатывается", "доставляется", "доставлен")     !!!!!!!!!!!

Удалить заказ
DELETE /orders/:id      !!!!!!!


1. Маршрут для создания подкатегории (POST /categories/:categoryId/subcategories)
Создаёт подкатегорию, связанную с определённой категорией, переданной через categoryId.

2. Маршрут для получения подкатегорий категории (GET /categories/:categoryId/subcategories)
Возвращает список подкатегорий для конкретной категории.

3. Маршрут для обновления подкатегории (PUT /categories/:categoryId/subcategories/:subCategoryId)
Обновляет подкатегорию, используя subCategoryId и categoryId.

4. Маршрут для удаления подкатегории (DELETE /categories/:categoryId/subcategories/:subCategoryId)
Удаляет подкатегорию по её ID. //all done
